# 🌿 Dara Bragança Massoterapeuta

Uma landing page moderna, responsiva e otimizada para profissionais da área de bem-estar e massoterapia. Desenvolvida com foco em **conversão**, **experiência do usuário** e **design clean**, permite que clientes conheçam os serviços, visualizem benefícios e agendem sessões diretamente pelo WhatsApp.

---

## ✨ Funcionalidades

- ✅ **100% Responsivo** – Adapta-se perfeitamente a mobile, tablet e desktop
-  **Integração com WhatsApp** – Botão flutuante e CTAs com mensagem pré-preenchida
- 🎠 **Carrossel Interativo** – Autoplay, navegação por setas, indicadores e suporte a swipe
- 🎨 **CSS Variables** – Paleta de cores centralizada para fácil personalização
- ⚡ **Zero Dependências** – HTML, CSS e JS puros (apenas Google Fonts)
-  **Animações Suaves** – Scroll behavior, fade-ins e hover effects elegantes
-  **SEO Básico** – Tags semânticas, meta viewport e estrutura acessível

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia       | Uso                              |
|------------------|----------------------------------|
| HTML5            | Estrutura semântica da página    |
| CSS3             | Estilização, Grid, Flexbox, Variáveis, Animações |
| JavaScript       | Carrossel, interações e controle de autoplay |
| Google Fonts     | Cormorant Garamond & Jost        |
